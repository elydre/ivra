# ivra
interactive virtual rgt

## Update Note

v1 is in progress, if you are looking for v0 (in c++) watch [the releases](https://github.com/elydre/ivra/releases)

## Instructions

```
00.   DIS => display R(1)
01.   SET => R(1) = (2)
02.   CPY => R(1) = R(2)
03.   TRC => R(1) = R(R(2))

04.   ADD => R(1) = R(1) + R(2)
05.   SUB => R(1) = R(1) - R(2)
06.   MUL => R(1) = R(1) * R(2)
07.   DIV => R(1) = R(1) / R(2)

08.   AND => R(1) = R(1) & R(2)
09.   HOR => R(1) = R(1) | R(2)
10.   NOT => R(1) = ~R(1)

11.   SUP => R(1) = R(1) > R(2)
12.   EQU => R(1) = R(1) == R(2)

13.   JMP => PC = R(1)
14.   HLT => exit
```
