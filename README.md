# ivra
interactive virtual rgt

## Update Note

v1 is in progress, if you are looking for v0 (in c++) watch [the releases](https://github.com/elydre/ivra/releases)

## Instructions

```
0.   DIS => display R(1)
1.   SET => R(1) = (2)
2.   CPY => R(1) = R(2)
3.   TRC => R(1) = R(R(2))

4.   ADD => R(1) = R(1) + R(2)
5.   SUB => R(1) = R(1) - R(2)
6.   MUL => R(1) = R(1) * R(2)
7.   DIV => R(1) = R(1) / R(2)

8.   AND => R(1) = R(1) & R(2)
9.   HOR => R(1) = R(1) | R(2)
A.   NOT => R(1) = ~R(1)

B.   SUP => R(1) = R(1) > R(2)
C.   EQU => R(1) = R(1) == R(2)

D.   JMP => PC = R(1)
```
